* Local Development Setup

#+BEGIN_SRC bash
# Clone
git clone https://github.com/jwalsh/cprr.git
cd cprr

# Create .env from template
gmake .env

# Edit .env for your platform (see "Platform Notes" below)
# Then enable direnv
direnv allow

# Build
gmake cprr

# Run locally
./cprr init --examples
./cprr list

# Run tests
gmake test

# Install to $GOPATH/bin
gmake install
#+END_SRC

** PATH Setup

=gmake install= puts the binary in =~/go/bin/= (standard Go semantics).
Ensure your PATH includes user-local bin directories.

Add to =~/.profile=:

#+BEGIN_SRC bash
export PATH=$HOME/go/bin:$HOME/.local/bin:$PATH
#+END_SRC

Verify after sourcing:

#+BEGIN_SRC bash
. ~/.profile
which cprr        # → ~/go/bin/cprr
cprr --version
#+END_SRC

** Environment Configuration

The project uses =direnv= to load =.env= into your shell:

#+BEGIN_SRC
.env.example   → Template with documented options (committed)
.env           → Local overrides (gitignored)
.envrc         → Loads .env via dotenv_if_exists (committed)
#+END_SRC

Key variables:

| Variable          | Default | Description                       |
|-------------------+---------+-----------------------------------|
| =GO=              | =go=    | Go binary name (see platform notes) |
| =CPRR_DATA_DIR=   | -       | Override default data location    |
| =CPRR_CI=          | -       | Set =true= in CI environments    |

** Platform Notes

*** FreeBSD

Go is installed as a versioned binary (e.g., =go124=):

#+BEGIN_SRC bash
# Check installed version
pkg info | grep go

# Edit .env
echo "GO=go124" >> .env
direnv allow
#+END_SRC

*** Linux / macOS

The default =GO=go= should work. No =.env= changes needed.

* Manual Testing Checklist

#+BEGIN_SRC bash
# Clean slate
rm -rf ~/.cprr .cprr

# Test init
./cprr init --examples
./cprr list

# Test state machine guards
./cprr add "No hypothesis"
./cprr next 4                    # Should fail: missing hypothesis

./cprr add "With hypothesis" --hypothesis "Expected outcome"
./cprr next 5                    # Should succeed: open → testing
./cprr next 5                    # Should fail: missing evidence

./cprr evidence 5 "First observation"
./cprr next 5                    # Should fail: need 2+ evidence

./cprr evidence 5 "Second observation"
./cprr next 5                    # Should succeed: testing → confirmed

# Test dry-run
./cprr status 5 testing
./cprr next 5 --dry-run

# Test force bypass
./cprr status 5 open
./cprr next 5 --force            # Bypasses hypothesis guard

# Cleanup
rm -rf .cprr
#+END_SRC

* Commit Guidelines

Use [[https://www.conventionalcommits.org/][conventional commits]]:

#+BEGIN_SRC
feat: add new command
fix: correct state transition bug
docs: update README
refactor: extract guard logic
test: add state machine tests
chore: update dependencies
#+END_SRC

Include co-author trailer:

#+BEGIN_SRC bash
git commit -m "feat: add refute command" \
  --trailer "Co-Authored-By: Your Name <email@example.com>"
#+END_SRC

* Adding New Guards

Guards are defined in =main.go=:

#+BEGIN_SRC go
var transitions = []Transition{
    {
        From: StatusTesting,
        To:   StatusConfirmed,
        Guards: []Guard{
            {
                Name:    "min_evidence",
                Check:   func(c *Conjecture) bool { return len(c.Evidence) >= 2 },
                Message: "minimum 2 pieces of evidence required",
            },
        },
    },
}
#+END_SRC

To add a new guard:

1. Define the =Check= function (returns =true= if guard passes)
2. Add descriptive =Name= and =Message=
3. Add to appropriate transition's =Guards= slice
4. Test with =cprr next <id>= to verify violation message

* Adding New Commands

1. Add case to =main()= switch
2. Implement =cmdFoo(store *Store, args []string)=
3. Update =usage()= help text
4. Add to README.org and this file

* Worktrees

Parallel work uses git worktrees under =worktrees/= (gitignored):

#+BEGIN_SRC bash
# List worktrees and sync with remote
gmake worktrees ARGS=sync

# Create a worktree for a feature branch
gmake worktrees ARGS="add feat/CPRR-xxx-description"

# Prune stale worktrees
gmake worktrees ARGS=prune
#+END_SRC

See =AGENTS.md= for the full worktree workflow.
