* Creating a New Experiment

#+BEGIN_SRC bash
# Get next number
NEXT=$(printf "%03d" $(($(ls -d [0-9]* 2>/dev/null | wc -l) + 1)))

# Create structure
mkdir -p ${NEXT}-my-hypothesis
cd ${NEXT}-my-hypothesis

# Create CONJECTURE.md with hypothesis, motivation, falsification criteria
# Then implement, gather evidence, attempt refutation
#+END_SRC

* Workflow

1. *CONJECTURE* — State hypothesis with falsification criteria
2. *PROOF* — Gather evidence supporting the hypothesis
3. *REFUTATION* — Actively try to disprove (intellectual honesty)
4. *REFINEMENT* — Adjust scope based on what was learned

* Refuted Experiments

Refuted experiments are *not deleted*. They represent negative knowledge —
things we tried that didn't work. Search them before proposing new ideas:

#+BEGIN_SRC bash
grep -r "Status.*REFUTED" experiments/*/CONJECTURE.md
#+END_SRC
