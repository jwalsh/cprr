This directory contains the source materials for the CPRR SPLASH 2026 paper,
authored in org-mode with ACM =sigconf= LaTeX export.

* Build Targets

| Target     | Command          | Output                              |
|------------+------------------+-------------------------------------|
| Paper      | =make paper=     | =cprr-splash.pdf= (ACM 2-column)   |
| Sprint     | =make sprint=    | =.html .md .txt .tex .pdf=         |
| Diagrams   | =make diagrams=  | =.png= from =.mmd= (needs =mmdc=) |
| Everything | =make all=       | All of the above                    |
| Clean      | =make clean=     | Remove =.aux .log .bbl= artifacts  |
| Nuke       | =make distclean= | Remove all generated files          |

* Notes

- The =.tex= was initially generated via =ox-latex= then hand-tuned for ACM
  compliance. Both the =.org= and =.tex= are maintained; the =.org= is the
  canonical source for content, the =.tex= for submission.
- =ulem.sty= may be missing from minimal texlive installs. If =ox-latex=
  inserts =\usepackage[normalem]{ulem}=, comment it out unless you use
  strikethrough formatting.
- The Mermaid diagrams render in any browser via =cprr-diagrams.html= (loads
  mermaid.js from CDN). For LaTeX inclusion, render to PNG locally with =mmdc=.
